Blinkit Sales Analysis

This project analyzes Blinkit sales data to identify key trends across items, outlets, and customer ratings.
The dataset was cleaned and explored using SQL, Python, and Power BI to calculate important KPIs and create meaningful visual insights and dashboards.

Project Goals:
Clean and prepare the raw Blinkit dataset
Identify sales patterns across product categories and outlet types
Build KPIs for business understanding
Create visual charts and a dashboard for quick insights

Tools & Technologies Used:
SQL for Data cleaning, transformations & KPI calculations
Python (Pandas, Matplotlib/Seaborn) for EDA & visual analysis
Power BI for Interactive dashboard
Jupyter Notebook For writing and running analysis code
